### Node.js SDK

The Node.js TypeScript SDK for GoBetterAuth provides a seamless way to interact with the GoBetterAuth server.

With support for client-side plugins, it gives you powerful tools to enhance your authentication workflows and interact with your GoBetterAuth server in a flexible and extensible manner.

You can view the SDK here: [Node.js SDK](https://www.npmjs.com/package/go-better-auth)
