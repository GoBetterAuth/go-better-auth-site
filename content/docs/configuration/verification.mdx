## Verification Configuration

Configure verification token storage:

```go
Verification: &domain.VerificationConfig{
    ModelName: "verification",
    Fields: map[string]string{
        "token": "verification_token",
    },
    DisableCleanup: false,
}
```

## Token Types

The verification system handles multiple token types:

- **Email verification tokens** for new user verification
- **Password reset tokens** for secure password changes
- **Email change tokens** for updating user email addresses
- **Account deletion tokens** for secure account removal

## Automatic Cleanup

By default, expired verification tokens are automatically cleaned up:

```go
DisableCleanup: false  // Enable automatic cleanup (recommended)
```

## Custom Field Mapping

Map verification table fields to custom column names:

```go
Fields: map[string]string{
    "token": "verification_token",
    "expiresAt": "expires_at",
    "identifier": "user_identifier",
}
```
