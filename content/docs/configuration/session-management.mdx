## Session Configuration

Customize session behavior:

```go
Session: &domain.SessionConfig{
    ModelName:             "session",
    Fields: map[string]string{
        "userId": "user_id",  // Custom field mapping
    },
    ExpiresIn:             7 * 24 * time.Hour,  // 7 days
    UpdateAge:             24 * time.Hour,      // 1 day
    DisableSessionRefresh: false,
    AdditionalFields: map[string]domain.AdditionalField{
        "device": {Type: "string"},
    },
    StoreSessionInDatabase:    true,
    PreserveSessionInDatabase: false,
    CookieCache: &domain.CookieCacheConfig{
        Enabled: true,
        MaxAge:  300,  // 5 minutes
    },
}
```

## Session Security

Sessions are stored securely with the following features:

- **Automatic expiration** based on `ExpiresIn`
- **Sliding expiration** with `UpdateAge` for active users
- **Secure cookies** with HTTPOnly and SameSite attributes
- **Database persistence** for server-side session storage
- **Cookie caching** for performance optimization

## Token Refresh

The library supports automatic token refresh flows. When a session is near expiration, clients can request new tokens without re-authentication.

## Custom Fields

Add additional fields to session records:

```go
AdditionalFields: map[string]domain.AdditionalField{
    "ipAddress": {Type: "string"},
    "userAgent": {Type: "string"},
    "deviceId":  {Type: "string"},
}
```
