Follow these steps to integrate Discord OAuth 2.0 authentication into your application using the `DiscordProviderConfig`.

<Steps>
<Step>

### Register Your Application on Discord

- Go to the [Discord Developer Portal](https://discord.com/developers/applications).
- Create a new application or select an existing one.
- Navigate to the "OAuth2" section in the sidebar.
- Under "Redirects", add your redirect URI (e.g., `http://localhost:8080/api/auth/discord/callback`).

</Step>

<Step>

### Obtain Your Credentials

In the OAuth2 section, copy the **Client ID** and **Client Secret**.
Use these values to configure your `DiscordProviderConfig`.

</Step>

<Step>

### Configure the Provider

Use the following Go struct to hold your Discord OAuth configuration:

```go
auth, err := gobetterauth.New(&gobetterauthdomain.Config{
  // ...other options
  SocialProviders: &gobetterauthdomain.SocialProvidersConfig{
    Discord: gobetterauthdomain.DiscordProviderConfig{
      ClientID:               os.Getenv("DISCORD_CLIENT_ID"),
      ClientSecret:           os.Getenv("DISCORD_CLIENT_SECRET"),
      SocialProviderOptions:  gobetterauthdomain.SocialProviderOptions{
        RedirectURI: "http://localhost:8080/api/auth/callback/discord",
      },
		},
  },
})
```

</Step>
</Steps>
